## Competitive Advantage

1. **Home-centric model** - Data stays with property
2. **Multi-user ecosystem** - Owners + contractors + realtors
3. **AI/AR future-ready** - Not just a database
4. **Privacy-first** - No data selling
5. **Trades expertise** - Built by a Red Seal carpenter

## Risk Mitigation

- Start with homeowner focus, add B2B later
- Freemium model reduces acquisition friction
- Modular architecture allows pivots
- AI-assisted development keeps costs low
- Focus on boring, reliable tech (not bleeding edge)
