# HOOOMZ INTERIORS — JOB FLOW HIERARCHY
## Structural Foundation Document v1.0

---

## ARCHITECTURE DECISIONS

- **One process, three lenses.** The job moves through one pipeline. Manager, Operator, and Homeowner each see a filtered view of the same job.
- **Process is built from the Manager's perspective.** Every stage, status, and decision point is defined by what the Manager needs to see, do, and know.
- **Stage-first with trade breakdown inside Active (Option C).** Stages own the front and back end. Trades own the execution middle.
- **Four core trades:** Flooring, Paint, Trim, Doors.
- **No Looops sphere visuals.** Interiors has its own identity — construction-confident, royal blue/near-black, clean and professional.

---

## STAGE OVERVIEW

| # | Stage | Status Bucket | Description |
|---|-------|--------------|-------------|
| 1 | Lead | Pipeline | First contact made, no commitment |
| 2 | Consultation | Pipeline | Site visit booked or completed |
| 3 | Estimate | Pipeline | Proposal built and sent |
| 4 | Booked | Scheduled | Deposit collected, start date confirmed |
| 5 | Prep | Scheduled | Materials ordered, crew assigned |
| 6 | Active | Active | Work is happening (trade breakdown) |
| 7 | Punch | Punch/Close | Work complete, deficiencies resolved |
| 8 | Turnover | Punch/Close | Walkthrough, Home Care Sheet, final invoice |
| 9 | Post-Turnover | Complete | Warranty period, maintenance bridge |

## STAGE 1 — LEAD

*First contact. No commitment yet. Source tracked.*

### Manager
- [ ] Log homeowner name and contact info
- [ ] Record lead source (Ritchies referral / Home Show / website / word of mouth / referral)
- [ ] Note project type and rough scope
- [ ] Assign consultation to self or operator

### Operator
- [ ] Schedule consultation
- [ ] Send confirmation to homeowner (date, time, what to expect)

### Homeowner
- Receives: Consultation confirmation

---

## STAGE 2 — CONSULTATION

*Site visit. Measurements, photos, scope discussion.*

### Manager
- [ ] Conduct site visit
- [ ] Take room-by-room measurements
- [ ] Capture before-state photos
- [ ] Assess scope and complexity
- [ ] Identify concerns (subfloor condition, existing trim condition, moisture, access)
- [ ] Discuss trade scope, materials, and finishes with homeowner
- [ ] Set expectation on estimate turnaround

### Operator
- [ ] Prepare consultation package (intake form, measurement sheet)
- [ ] Log consultation notes into job record
- [ ] Book estimate delivery follow-up

### Homeowner
- Available for walkthrough
- Shares vision and preferences
- Receives: Consultation summary / what happens next

---

## STAGE 3 — ESTIMATE

*Proposal built and sent. Homeowner reviewing.*

### Manager
- [ ] Build estimate by trade (Flooring / Paint / Trim / Doors)
- [ ] Review for scope accuracy and margin
- [ ] Approve estimate for delivery

### Operator
- [ ] Format and send proposal to homeowner
- [ ] Follow up if no response within 3 business days
- [ ] Log homeowner response (accepted / declined / questions)

### Homeowner
- Reviews proposal
- Asks questions
- Accepts or declines
- Receives: Proposal document, follow-up if no response

---

## STAGE 4 — BOOKED

*Proposal accepted. Deposit collected. Job is real.*

### Manager
- [ ] Confirm final scope with homeowner (no verbal-only changes)
- [ ] Collect deposit
- [ ] Confirm start date and access arrangements

### Operator
- [ ] Create formal job record
- [ ] Issue booking confirmation to homeowner
- [ ] Create materials list by trade
- [ ] Schedule crew by trade
- [ ] Block calendar

### Homeowner
- Signs proposal / contract
- Pays deposit
- Confirms access arrangements
- Receives: Booking confirmation, start date, what to expect

---

## STAGE 5 — PREP

*Materials ordered. Crew briefed. Job is ready to execute.*

### Manager
- [ ] Review job plan (scope, crew, schedule)
- [ ] Confirm crew assignment per trade
- [ ] Pre-job site check if required (complex jobs or structural concerns)

### Operator
- [ ] Order all materials (with delivery dates confirmed)
- [ ] Verify materials received and correct before crew deployment
- [ ] Brief crew (scope, schedule, site conditions, homeowner notes)
- [ ] Prepare job package (photos, measurements, scope, checklists)
- [ ] Confirm homeowner access / schedule for day 1

### Homeowner
- Prepares space (clear furniture, remove personal items from work areas)
- Confirms access
- Receives: Pre-job communication (what to expect, day 1 logistics)

---

## STAGE 6 — ACTIVE

*Work is happening. Organized by trade. Each trade runs its own checklist.*

> **Note:** Trades may run concurrently or in sequence depending on job scope. Each trade has independent status tracking within the Active stage.

---

### TRADE: FLOORING

#### Shield (Site Protection)
- [ ] Door jambs undercut or protected
- [ ] Adjacent flooring transitions protected
- [ ] HVAC vents covered
- [ ] Furniture / appliances moved or noted in scope

#### Clear (Demo & Subfloor Prep)
- [ ] Existing flooring removed (if in scope)
- [ ] Subfloor inspected — damage, soft spots, squeaks noted and reported
- [ ] Subfloor repairs completed (logged as change order if out of original scope)
- [ ] Floor level checked — tolerance confirmed within spec
- [ ] Subfloor clean and clear

#### Ready (Material Staging)
- [ ] Flooring material on site and acclimated (min 48 hrs LVP / 72 hrs hardwood)
- [ ] Underlayment on site
- [ ] Transition strips / thresholds on site
- [ ] Tools and adhesives staged

#### Install
- [ ] Underlayment installed per manufacturer spec
- [ ] Flooring direction confirmed (with homeowner or noted in plan)
- [ ] First row set and checked for square
- [ ] Field installed with proper expansion gaps maintained
- [ ] Transitions cut and staged (not installed until Punch)
- [ ] Progress photos taken at midpoint and rough completion
- [ ] Installer flags any issues to Operator before proceeding

---

### TRADE: PAINT

#### Shield (Site Protection)
- [ ] Floors protected (dropcloths / paper)
- [ ] Trim, windows, doors masked
- [ ] Furniture covered or removed
- [ ] Switch plates / outlet covers removed and stored

#### Clear (Surface Prep)
- [ ] Walls washed / deglossed where required
- [ ] Holes and cracks filled
- [ ] Patching sanded and primed
- [ ] Caulk joints inspected — gaps addressed before painting

#### Ready (Material Staging)
- [ ] Paint colors confirmed in writing (no verbal-only color changes)
- [ ] All paint on site, shaken, and labelled by room
- [ ] Brushes / rollers / sprayer ready

#### Install
- [ ] Primer coat applied where required (new drywall, stained surfaces)
- [ ] First coat applied — ceiling first, then walls
- [ ] First coat dry time observed (per product spec)
- [ ] Second coat applied
- [ ] Progress photos taken after each coat
- [ ] Touch-up areas noted on punchlist — not addressed until Punch stage

---

### TRADE: TRIM

#### Shield (Site Protection)
- [ ] Floors protected at work area
- [ ] Adjacent painted surfaces protected

#### Clear (Demo & Prep)
- [ ] Old trim removed (if in scope)
- [ ] Wall surface prepped at trim locations (patched, primed)
- [ ] Door jambs checked for plumb and level — issues reported before install

#### Ready (Material Staging)
- [ ] All trim material on site and matched to plan
- [ ] Correct profiles confirmed against spec
- [ ] Paint / stain finish confirmed

#### Install
- [ ] Baseboard installed
- [ ] Door casings installed
- [ ] Window casings installed
- [ ] Crown / ceiling trim installed (if in scope)
- [ ] All joints tight — coped or mitered correctly
- [ ] Nails set, holes filled
- [ ] Caulk applied at wall joints
- [ ] Progress photos taken

---

### TRADE: DOORS

#### Shield (Site Protection)
- [ ] Floor protected at door locations
- [ ] Adjacent trim and paint protected

#### Clear (Demo & Prep)
- [ ] Old doors / hardware removed (if in scope)
- [ ] Rough opening checked for square and plumb
- [ ] Any repairs to opening completed before hang (logged as change order if out of scope)

#### Ready (Material Staging)
- [ ] All doors on site
- [ ] All hardware on site (hinges, locksets, stops)
- [ ] Hinge count and style confirmed

#### Install
- [ ] Door hung and shimmed plumb / level
- [ ] Latch / strike aligned and operational
- [ ] Hardware installed and tested
- [ ] Door swing confirmed (matches plan)
- [ ] Gap margins consistent (3/32" standard all around)
- [ ] Progress photos taken

---

## STAGE 7 — PUNCH

*Work complete. Deficiencies identified and resolved before homeowner sees it.*

### Manager
- [ ] Full walkthrough of all four trades
- [ ] Deficiency list generated (written, not verbal)
- [ ] Deficiencies assigned to crew with due dates
- [ ] Return walkthrough — verify all deficiencies resolved
- [ ] Final quality sign-off

**Trade-specific punch items:**

**Flooring:**
- [ ] Transitions and thresholds installed
- [ ] Expansion gaps verified and consistent
- [ ] No lippage, gaps, or hollow spots
- [ ] All debris removed

**Paint:**
- [ ] All touch-ups complete
- [ ] No holidays, drips, or lap marks
- [ ] Masking removed cleanly
- [ ] Switch plates / outlet covers reinstalled

**Trim:**
- [ ] All caulk joints finished and smooth
- [ ] All nail holes filled and touched up
- [ ] No gaps at corners or joints
- [ ] Paint or stain finish complete on trim

**Doors:**
- [ ] All doors operate smoothly
- [ ] No binding, racking, or misalignment
- [ ] All hardware tight and functional
- [ ] Stops installed and adjusted

### Operator
- [ ] Schedule punch walkthrough
- [ ] Track deficiency status in job record
- [ ] Confirm all deficiencies resolved before Turnover is booked

### Homeowner
- Not involved until Stage 8. Homeowner sees the finished product, not the punchlist process.

---

## STAGE 8 — TURNOVER

*Walkthrough with homeowner. Home Care Sheet. Final invoice.*

### Manager
- [ ] Conduct walkthrough with homeowner
- [ ] Review Home Care Sheet line by line
- [ ] Address any homeowner questions
- [ ] Obtain homeowner sign-off on completion
- [ ] Confirm final invoice issued

### Operator
- [ ] Prepare Home Care Sheet (auto-generated from job record)
- [ ] Issue final invoice
- [ ] Collect final payment
- [ ] Archive job record and photos
- [ ] Send review / referral request within 24 hrs

### Homeowner
- Attends walkthrough
- Reviews Home Care Sheet
- Signs completion form
- Pays final invoice
- Receives: Home Care Sheet, review request, Post-Turnover contact info

---

## STAGE 9 — POST-TURNOVER

*Warranty period. Maintenance bridge.*

### Manager
- [ ] 30-day check-in scheduled at time of Turnover
- [ ] Any warranty issues logged and dispatched promptly
- [ ] Issues documented (photo, description, resolution)

### Operator
- [ ] 30-day follow-up sent (automated preferred)
- [ ] Maintenance package information sent
- [ ] Job closed in system after warranty period with no outstanding issues

### Homeowner
- Receives: 30-day check-in
- Receives: Maintenance package offer (bridge to Hooomz Maintenance division)

## NOTES FOR DEVELOPMENT

1. **Job record is the spine.** Every action, photo, note, and status change is an event logged against the job record. This is the Activity Log.

2. **Stage advancement is explicit.** A job does not auto-advance. Manager or Operator intentionally moves it forward. This prevents jobs drifting through stages without accountability.

3. **Checklists are gating.** Key checklists (especially Punch sign-off and Turnover sign-off) should require completion before stage advance is permitted.

4. **Change orders trigger at Clear.** When subfloor damage, rough opening issues, or other out-of-scope conditions are discovered, the workflow branches to a Change Order before work continues.

5. **Photos are required at defined milestones.** Not optional. Before state, midpoint, and completion photos are part of the job record and feed the Home Care Sheet.

6. **Home Care Sheet auto-generates from the job record.** Trade scope, materials used, care instructions, and warranty info pull from the job data.

7. **Homeowner view is always one stage behind.** They see what's done, not what's in progress. Protects quality perception.
